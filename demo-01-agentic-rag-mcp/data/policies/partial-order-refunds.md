# Partial Order Refunds

This policy covers refunding one or more individual items within a multi-item order while keeping the rest of the order. It applies specifically to itemized orders.

## Per-Item Calculation, No Percentage Cap

When a customer is refunded for specific items in a multi-item order, the refund is **calculated per item** — the actual price paid for each affected item, plus any tax attributed to that item. There is **no percentage cap** on a per-item refund: if the affected items happen to represent most or all of the order, the per-item total can be refunded accordingly, up to the original order amount.

This per-item approach is the correct method for multi-item orders. (Note: this differs from the flat percentage cap described in the partial refund policy, which addresses goodwill-style adjustments rather than itemized refunds.)

## How to Calculate

1. Identify which items the customer is keeping and which are being refunded.
2. Sum the price paid for each refunded item, including its share of tax.
3. Include shipping for a refunded item only if that item is being fully returned and shipping was itemized.
4. The total of the refunded items is the refund amount.

## Examples

- **Example A:** A 3-item order ($20 + $30 + $50). The customer returns only the $30 item. Refund = $30 (plus its tax).
- **Example B:** A 4-item order where 3 items are defective. Refund the sum of those 3 items' prices, even if that exceeds 50% of the order — the per-item method has no percentage cap.

The combined refund may never exceed the original order amount; see refund amount limits.
